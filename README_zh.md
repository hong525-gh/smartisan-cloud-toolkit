# 锤子便签批量删除助手

一个用于批量删除锤子便签（cloud.smartisan.com）的油猴脚本。

## 要解决的问题

锤子便签云端应用没有提供批量删除功能。逐条手动删除数百条便签耗时且繁琐。

## 解决方案

该脚本在锤子便签页面右下角添加一个悬浮按钮。它从浏览器的 IndexedDB 中读取你的便签数据，提供可搜索的选择面板，然后通过模拟点击 Web 应用自身的删除流程来自动执行删除——确保删除操作被正确同步到服务器。

## 功能特性

- 从本地 IndexedDB 读取全部便签（无需调用 API）
- 按标题搜索和过滤便签
- 可逐条选择、按文件夹选择或全选
- 可配置删除间隔以避免被限流
- 删除过程中可暂停 / 恢复 / 停止
- 实时进度面板

## 安装

1. 为浏览器安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 打开[脚本原始文件](https://github.com/hong525-gh/smartisan-notes-delete/raw/main/%E9%94%A4%E5%AD%90%E4%BE%BF%E7%AD%BE%E6%89%B9%E9%87%8F%E5%88%A0%E9%99%A4%E5%8A%A9%E6%89%8B.user.js) — Tampermonkey 会提示安装
3. 访问 <https://cloud.smartisan.com/#/notes> 并登录
4. 点击右下角的红色垃圾桶图标

## 使用方法

1. 点击红色悬浮按钮 → **批量删除…**
2. 搜索/过滤便签，勾选要删除的条目
3. 如需调整删除间隔（默认 4000 毫秒/条）
4. 点击 **确认删除** → 确认 → 脚本将自动通过 UI 逐条删除

## 工作原理

1. 从 `_pouch_note` 和 `_pouch_folder` 两个 IndexedDB 存储中读取便签数据
2. 展示选择面板供用户勾选便签
3. 对每条选中的便签：在锤子便签侧边栏中找到该便签、点击选中、点击删除按钮、确认弹窗——全部通过模拟点击完成，间隔可配置

## 安全性

- 默认每条间隔 4 秒，防止被限流
- 每删除 5 条额外休息 8 秒
- 删除过程中可暂停/停止
- 执行前需二次确认
